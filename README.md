# CRM - Ticket Module

This project is a CRM Ticket Management module built using PHP and MySQL.

## Features
- User Authentication (Register/Login)
- Ticket Creation
- Ticket Assignment
- Role-based Access Control
- Status Management

## Tech Stack
- PHP (Core PHP)
- MySQL
- Apache (WAMP)
- Git & GitHub

## Setup
1. Install WAMP.
2. Place project inside `www` directory.
3. Create database `crm_ticket`.
4. Import SQL from `database/migrations`.

More documentation will be added during development.
